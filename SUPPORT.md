# Support

## Getting Help

### Bug Reports
Found a bug? Please [open an issue](https://github.com/Anigx/Furya-Public/issues/new/choose) using the Bug Report template.

**Please include:**
- What happened
- Steps to reproduce
- App version
- Android version
- Device model

### Feature Requests
Have a feature idea? Please [open an issue](https://github.com/Anigx/Furya-Public/issues/new/choose) using the Feature Request template.

### Downloads
Official APK releases are available on the [GitHub Releases](https://github.com/Anigx/Furya-Public/releases) page.

### Updates
Furya can check for updates automatically. You can also manually check in Settings > Updates.

## What We Support

- Bug reports for the latest stable version
- Feature requests
- Questions about using Furya

## What We Don't Support

- Old Android versions (below Android 10)
- Unofficial APK builds
- Issues caused by third-party modifications
- Feature requests for major architectural changes

## Contributing
This repository does not accept source code contributions as Furya is closed-source. However, we welcome feedback and feature ideas through issues.
